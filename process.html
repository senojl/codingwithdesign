I initially started off with research into language learning, communication, and from there landed in colloquialisms. I defined these communications as “social subtleties that human interaction”. While I was interested in these concepts, I became fascinated with programming languages, which are human constructs designed to communicate with machines, and moves further from my idea of communication with others, and more into constructed logic and syntax built around computers

“Translation should not be done as a separate process; it should be an extension of the work being done.” Modern Translation Guide

After doing research in teaching designers Android programming, I realized that I wanted to take a different approach, and looked into an app that described design based esoteric programming languages. I narrowed my research down into five esoteric coding languages, and started working on visual language, and the structure of the app and accompanying website.

My concept for the esoteric language app was to have an app that displayed each language and an example of their code in the form of a Hello World program. Then the website would display the design for the app compared to the code for the app. This would serve as an example of coding and design, which became the name for the app.
